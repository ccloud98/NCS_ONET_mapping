  1. 프로젝트 개요
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  본 보고서는 한국의 NCS(국가직무능력표준) 사업관리 분야와 미국의 O*NET(직업
  정보 네트워크) 데이터를 매핑하기 위한 전처리 파이프라인의 실행 결과를 기록한
  문서입니다.

  ■ 목적
    - NCS 사업관리 능력단위 텍스트를 영어로 번역
    - O*NET 전체 직업별 설명 텍스트를 구조화
    - 두 데이터셋을 의미적 유사도 기반으로 매핑할 수 있는 기반 데이터 생성

  ■ 주요 파일 경로
    - 원본 NCS 데이터     : data/raw/NCS_DB.xlsx
    - 원본 O*NET 데이터   : data/raw/db_30_2_excel/
    - 전처리 결과 저장    : data/processed/
    - 전처리 노트북       : notebooks/01_data_preprocessing.ipynb

  사용 라이브러리:
  • pandas / numpy   : 데이터 조작 및 집계
  • pathlib.Path     : OS 독립적 파일 경로 관리
  • tqdm             : 번역 진행 상태 표시
  • deep_translator  : GoogleTranslator 기반 한→영 번역
  • warnings         : 불필요 경고 억제

    ../data/raw/           ← 원본 데이터 (읽기 전용)

  ┌─────────────────────────────────────────────────────────────────────────┐
  │ Step 2 : NCS 로딩 및 필터링 (Cell 2)                                   │
  └─────────────────────────────────────────────────────────────────────────┘

  처리 흐름:
    1) NCS_DB.xlsx 로드 (openpyxl 엔진, 한글 컬럼명 자동 인식)
    2) 중분류 컬럼 자동 탐지: '중분류'와 '명'이 모두 포함된 컬럼 검색
       → 탐지 결과 : '중분류코드명'
    3) 해당 컬럼 값이 '사업관리'를 포함하는 행만 필터링
       → 결과 : 9,089행 (DB 전체 = 모두 사업관리이므로 변화 없음)
    4) 소분류 컬럼 자동 탐지: '소분류'와 '명' 포함 컬럼 검색
       → 탐지 결과 : '소분류코드명'
    5) 결측값 fillna("") 일괄 처리

  설계 원칙:
    컬럼명을 하드코딩하지 않고 키워드 패턴 매칭으로 동적 탐지.
    향후 파일 구조가 변경되어도 재사용 가능.

  │ Step 3 : NCS 텍스트 합산 (Cell 3)                                      │
  └─────────────────────────────────────────────────────────────────────────┘

  텍스트 합산 대상 컬럼 (우선순위 순):
    소분류코드명 > 세분류코드명 > 능력단위명칭 > 능력단위요소명

  처리 흐름:
    1) find_col() 함수로 각 컬럼 동적 탐지
    2) 행별로 4개 컬럼 텍스트를 공백으로 이어 붙여 combined_text 생성
    3) 소분류코드명 기준으로 groupby → combined_text 합산
    4) ncs_full_text : 전체 소분류 텍스트를 단일 문자열로 합산

  │ Step 4 : O*NET 데이터 로딩 및 병합 (Cell 4)                            │

  처리 흐름:
    1) Occupation Data.xlsx 로드 → onet_occ (1,016행)
    2) Task Statements.xlsx 로드 → onet_tasks (18,796행)
    3) 컬럼명 양쪽 공백 제거 (.str.strip())
    4) Task 텍스트 컬럼 동적 탐지:

    7) Description 컬럼이 있으므로:
       final_text = task_text + " " + Description
